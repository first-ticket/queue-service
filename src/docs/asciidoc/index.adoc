== 개요

대기열 진입 / 조회 / 취소 API.

=== 인증

모든 API 는 Gateway 가 검증한 `X-User-Id` 헤더를 사용한다.

=== 응답 형식

성공 / 실패 모두 `ApiResponse<T>` 형식으로 응답한다.

[source,json]
----
{
  "success": true,
  "code": "QUEUE_TOKEN_ISSUED",
  "message": "대기 토큰이 발급되었습니다",
  "timestamp": "2026-04-30T12:00:00",
  "data": {
    "tokenId": "...",
    "status": "WAITING",
    "issuedAt": "...",
    "position": 50
  }
}
----
